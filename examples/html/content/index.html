Sindarin HTTP Server

A simple static file server written in Sindarin

Welcome

This page is being served by a Sindarin HTTP server. The server reads static files from the content directory and serves them with the appropriate content type.

Features

  • Static file serving
  • Automatic content-type detection
  • CSS stylesheet support
  • Clean URL routing

How It Works

The server uses the sindarin-pkg-http package to handle HTTP requests and responses. When you request a file, it:

  1. Checks if the file exists in the content directory
  2. Determines the content type from the file extension
  3. Reads the file and returns it with proper headers
